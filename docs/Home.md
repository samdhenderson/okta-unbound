## What is Okta Unbound?

Okta Unbound is a Chrome extension that streamlines Okta administration with powerful bulk operations, intelligent filtering, and automated cleanup tools. It helps administrators manage large groups efficiently, maintain security posture, and perform operations that would otherwise take hours.

## Key Features

- **Security Posture Analysis** - Detect orphaned accounts, inactive users, and security risks
- **Dashboard Analytics** - Real-time insights with visual charts and metrics
- **Multi-Group Operations** - Bulk operations across multiple groups simultaneously
- **Audit Trail & Compliance** - Complete operation history for SOC2 compliance
- **Smart Automation** - One-click cleanup of inactive users
- **Export Capabilities** - Export member lists and reports to CSV/JSON
